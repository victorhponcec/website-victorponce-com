Proyectos Destacados

Portafolio: Diseño e Implementación de un Entorno Seguro en AWS

Este proyecto se enfoca en una arquitectura segura en AWS, construida con Terraform (IaC), integrando controles de defensa en profundidad, en redes, identidad, monitoreo y en las distintas capas de la aplicación.


La solución base consiste en una aplicación de 3 capas (Web, App y base de datos) con fuertes medidas de seguridad: subredes privadas, reglas estrictas en Grupos de Seguridad, logging centralizado, detección de amenazas, protección en el borde y capa 7, revisiones automáticas de cumplimiento, entre otros. Todo esto alineado con el marco AWS Well-Architected.


El proyecto se centra en la aplicación de principios de seguridad utilizando servicios nativos de AWS. El código utiliza Terraform para desplegar la infraestructura de manera inmutable y auditable, siguiendo las mejores prácticas de DevSecOps.
