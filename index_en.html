Featured Projects

Portfolio: Design and Implementation of a Secure Environment in AWS

This project focuses on a secure architecture in AWS, built with Terraform (IaC), integrating defense-in-depth controls across networking, identity, monitoring, and the different application layers.


The base solution consists of a 3-tier application (Web, App, and database) with strong security measures: private subnets, strict Security Group rules, centralized logging, threat detection, edge and layer 7 protection, automatic compliance reviews, among others. All aligned with the AWS Well-Architected framework.


The project centers on applying security principles using native AWS services. The code uses Terraform to deploy infrastructure immutably and auditable, following DevSecOps best practices.
